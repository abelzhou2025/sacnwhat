# 📱 快速移动端测试指南

## 🚀 3 步开始测试

### 步骤 1: 启动开发服务器

```bash
npm run dev
```

或者使用 Netlify Dev（包含 Functions）:

```bash
npm run dev:netlify
```

### 步骤 2: 打开浏览器并启用移动模式

1. 访问: http://localhost:3000 (或 Netlify Dev 显示的 URL)
2. 按 `F12` 打开开发者工具
3. 点击设备图标 📱 或按快捷键:
   - Mac: `Cmd + Shift + M`
   - Windows: `Ctrl + Shift + M`

### 步骤 3: 选择设备并测试

1. **选择设备预设**
   - iPhone 12/13/14 Pro
   - Pixel 5
   - 或自定义尺寸

2. **测试核心功能**
   - 📸 点击"Take Photo"测试拍照
   - 📤 点击"Upload Image"测试上传
   - 🔄 选择多张图片测试批量处理
   - 📝 查看 OCR 提取结果
   - 🗑️ 右滑删除历史记录

## 🎯 重点测试项

### ✅ 必须测试
- [ ] 拍照功能（移动端核心功能）
- [ ] 图片上传（单张和多张）
- [ ] OCR 文本提取
- [ ] 扫描历史显示
- [ ] 右滑删除功能

### 🔍 检查项
- [ ] 按钮大小是否适合触摸
- [ ] 文字是否清晰可读
- [ ] 布局在不同屏幕尺寸下是否正常
- [ ] 加载状态是否清晰
- [ ] 错误提示是否友好

## 💡 测试技巧

### 查看 Console 日志
- 打开开发者工具 → Console 标签
- 查看所有操作日志和错误

### 模拟慢网络
- 开发者工具 → Network 标签
- 选择 "Slow 3G" 或 "Fast 3G"
- 测试加载性能

### 测试不同屏幕
- 切换不同设备预设
- 测试横屏/竖屏
- 自定义屏幕尺寸

## 🐛 常见问题

**Q: 拍照按钮不工作？**
- 检查是否在移动模式下
- 查看 Console 是否有错误

**Q: 图片上传失败？**
- 检查图片大小
- 查看 Network 标签的请求

**Q: OCR 不工作？**
- 确保使用 `npm run dev:netlify`（包含 Functions）
- 检查 API 密钥是否设置
- 查看 Console 错误信息
